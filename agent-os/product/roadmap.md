# Product Roadmap

## Phase 1: MVP (All 7 Milestones)

All milestones are required for launch. Milestone 01 (Shell) is a dependency for all others; milestones 02-07 are recommended in order but can parallelize after Shell.

| # | Milestone | Scope |
|---|---|---|
| 01 | **Shell** | App layout, sidebar, routing, i18n (ES/EN), dark/light mode, auth flow |
| 02 | **Dashboard** | Role-specific stat cards, activity feed, quick actions |
| 03 | **Attendance** | Group roster, daily check-in, attendance reports, student history |
| 04 | **People** | User/group CRUD, role assignment, bulk operations |
| 05 | **Blog** | Posts, categories, emoji reactions, threaded comments, draft approval |
| 06 | **Teaching Tools** | Categorized utility grid, favorites, search |
| 07 | **AI Transcription** | Audio recorder, file upload, AI transcript + summary + action items |

### Role Visibility

- **Admin & Teacher:** All 6 sections
- **Student:** Dashboard, Attendance, Blog, Teaching Tools (no People, no AI Transcription)

## Phase 2: Post-Launch

To be determined. Focus is on launching and stabilizing all 7 milestones first.
