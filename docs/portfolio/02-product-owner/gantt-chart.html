Introducción

El presente diagrama de Gantt muestra la planificación temporal del proyecto Sistema Escolar CEICAVS, organizado en cinco sprints de dos semanas cada uno. El documento fue elaborado en mi rol de Product Owner y Team Lead para comunicar el alcance, las dependencias entre tareas y los hitos del proyecto a todos los stakeholders.

Cada sprint agrupa un conjunto de funcionalidades relacionadas que fueron diseñadas, desarrolladas y probadas dentro del periodo correspondiente. El diagrama permite visualizar el progreso acumulado y verificar que el proyecto se completó dentro del plazo planificado.

Notas Metodológicas

Metodología Scrum adaptada
El proyecto se desarrolla bajo una metodología ágil Scrum con sprints de dos semanas. Cada sprint concluye con una revisión del entregable y, cuando aplica, una demo ante el cliente (Product Owner). El seguimiento se realiza en Linear para el registro de tickets y GitHub para el control de versiones.
